# Парсер финансовых данных с Smart-Lab
Этот проект представляет собой веб-приложение для автоматического сбора финансовых данных по российским акциям с сайта Smart-Lab и сохранения их в формате JSON через локальный сервер.

## Основные возможности
- Автоматический парсинг финансовых данных по 100+ российским акциям
- Локальный сервер для приёма и сохранения данных в формате JSON
- Веб-интерфейс для запуска процесса сбора данных
- Скачивание готовых файлов с финансовыми отчётами
- Поддержка прокси для обхода ограничений CORS

## Требования
Node.js версии 14 или выше

## Запуск сервера
```bash
node server.js
Сервер запустится по адресу: http://localhost:3000/
```

## Использование
### Автоматический сбор данных
Откройте index.html в браузере

Скрипт автоматически начнёт собирать данные по всем акциям из списка

На странице появятся ссылки для скачивания каждого JSON-файла

Данные сохранятся в папке files/ на сервере

### Ручное скачивание данных
После завершения сбора данных, вы можете скачать любой файл по ссылке:

```
http://localhost:3000/files/НАЗВАНИЕ_АКЦИИ.json
```

Например:
```
http://localhost:3000/files/SBER.json

http://localhost:3000/files/GAZP.json
```
Формат данных
Каждый JSON-файл содержит финансовый отчёт в следующем формате:

```json
{
  "table": [
    {
      "ticket": "SBER",
      "report": {
        "Выручка": {
          "2020": "100 000",
          "2021": "120 000",
          "2022": "140 000"
        },
        "Чистая прибыль": {
          "2020": "50 000",
          "2021": "60 000",
          "2022": "70 000"
        }
        // ... другие финансовые показатели
      }
    }
  ]
}
```
## Настройка
Изменение списка акций
Отредактируйте массив stocks в файлах parser.js и test.js:

```javascript
const stocks = [
  "GAZP", "SBER", "LKOH",
  // ... добавьте или удалите тикеры
];
```
###Настройка сервера
В файле server.js можно изменить параметры сервера:

```javascript
const host = "localhost"  // Измените на нужный хост
const port = "3000"       // Измените на нужный порт
```
